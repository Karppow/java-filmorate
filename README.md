# Filmorate

Filmorate — backend-сервис для оценки фильмов и социальной активности пользователей.

Платформа позволяет ставить оценки фильмам, добавлять друзей и получать рекомендации.

---

## Архитектура

REST API приложение с классической многослойной структурой:

- Controller
- Service
- Repository

---

## Функциональность

### Пользователи
- создание профиля
- добавление друзей
- удаление друзей

### Фильмы
- добавление фильмов
- обновление информации
- удаление фильмов

### Рейтинги
- лайки фильмам
- популярные фильмы
- рекомендации

---
